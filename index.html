HyperText Markup Language

(Lingaguem de marcação de HiperTexto)

O html diferente do que muitos pensam, não é uma linguagem de programação e sim uma linguagem de marcação inventado por Tim Berners-Lee.

O Hipertexto tem como significado o texo por referencia de outros textos que se alternam(Links), linkando paginas por Hipertexto.

O html é o responsavel pela estrutura basica de uma web, é por ela que temos as primeiras impressões de uma página Web.

O que sãoTags?

As Tags são um conjunto de caracteres que formam um elemento no html, São formadas por um sinal de menor "<", o nome do elemento e um sinal de maior ">"
e para fechamento repetimos a estrutura, colocando um "/" antes do nome do elemento.

A também Tags que não possuem uma tag de fechamento, denominadas Tags vazias.

Exemplos de Tags e Tags de fechamento
  • <.p> e <./p>
  • <.h1> e <./h1>até h6
  • <.input>
  • <.img>

Atributos

Os atributos são informações especificas que damos as tags para que aja uma ação de acordo com o que foi informado no atributo.

Vale também falar sobre os Types(tipo) que acompanham os atributos. O comando type tem como objetivo especificar qual o tipo do atributo,
por exemplo a tag "<.input>" que ao adicionamos tipo por exemplo o "type="text" que define que o atributo é do tipo texto.
